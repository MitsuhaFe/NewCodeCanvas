# UI 设计规范文档

## 设计理念

### 核心原则

1. **简洁美观**：去繁就简，注重视觉层次
2. **高效易用**：减少操作步骤，提高效率
3. **现代化**：符合现代设计趋势
4. **一致性**：统一的视觉语言和交互模式
5. **性能优先**：流畅的动画和响应

### 设计目标

- **直觉性**：无需学习即可上手
- **愉悦感**：精致的细节和动画
- **专业性**：高质量的视觉呈现

## 组件设计

### 按钮 (Button)

#### 变体

**Primary（主要按钮）**：
```scss
.btn-primary {
  background: $primary;
  color: white;
  border: none;
  padding: 0.5rem 1rem;
  border-radius: $radius-md;
  font-size: $text-sm;
  font-weight: $font-medium;
  cursor: pointer;
  transition: all 0.15s ease;
  
  &:hover {
    background: $primary-dark;
    transform: translateY(-1px);
    box-shadow: $shadow-md;
  }
  
  &:active {
    transform: translateY(0);
  }
  
  &:disabled {
    background: $gray-600;
    cursor: not-allowed;
    opacity: 0.5;
  }
}
```

**Secondary（次要按钮）**：
```scss
.btn-secondary {
  background: transparent;
  color: $text-primary;
  border: 1px solid $border-primary;
  
  &:hover {
    background: $bg-tertiary;
    border-color: $gray-600;
  }
}
```

**Ghost（幽灵按钮）**：
```scss
.btn-ghost {
  background: transparent;
  color: $text-secondary;
  border: none;
  
  &:hover {
    background: rgba(255, 255, 255, 0.05);
    color: $text-primary;
  }
}
```

#### 尺寸


## 动画规范

### 动画时长

```scss
// 时长
$duration-fast: 150ms;
$duration-normal: 250ms;
$duration-slow: 350ms;

// 缓动函数
$ease-in: cubic-bezier(0.4, 0, 1, 1);
$ease-out: cubic-bezier(0, 0, 0.2, 1);
$ease-in-out: cubic-bezier(0.4, 0, 0.2, 1);
$ease-bounce: cubic-bezier(0.34, 1.56, 0.64, 1);
```

### 常用动画

```scss
// 淡入淡出
@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}

@keyframes fadeOut {
  from { opacity: 1; }
  to { opacity: 0; }
}

// 滑入滑出
@keyframes slideInUp {
  from {
    opacity: 0;
    transform: translateY(20px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

// 缩放
@keyframes scaleIn {
  from {
    opacity: 0;
    transform: scale(0.95);
  }
  to {
    opacity: 1;
    transform: scale(1);
  }
}

// 弹跳（Dock 图标）
@keyframes bounce {
  0%, 100% { transform: translateY(0) scale(1); }
  50% { transform: translateY(-20px) scale(1.1); }
}
```

## 无障碍规范

### ARIA 标签

```html
<!-- 按钮 -->
<button aria-label="关闭">
  <Icon name="x" />
</button>

<!-- 输入框 -->
<input
  type="text"
  aria-label="搜索壁纸"
  aria-describedby="search-help"
/>
<span id="search-help">输入关键词搜索</span>

<!-- 模态框 -->
<div
  role="dialog"
  aria-modal="true"
  aria-labelledby="modal-title"
>
  <h2 id="modal-title">设置</h2>
</div>
```

- Tab: 焦点移动

- 方向键: 列表/菜单导航
